# Chip-8 Emulator

## Controls

The controls follow a standard mapping and are mapped to the following keys:

```
1 2 3 4
q w e r
a s d f
z x c v
```

It's up to the specific games for what action each key performs.

## Usage

Click `Browse...` to upload a Chip-8 ROM from your computer.

Have fun!
